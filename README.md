# Advanced-Anticheat
A anti-cheat system for Roblox made to detect movement exploits, unauthorized remote execution, and environment tampering.

# How to Setup?
1. Make a folder named Libraries inside ServerScriptService and add the ProfileStore ModuleScript from https://github.com/MadStudioRoblox/ProfileStore inside it. This step can be skipped if you already have ProfileStore inside your game just change the ProfileStore Variable inside AntiCheatData to the area where your ProfileStore is and require it.
